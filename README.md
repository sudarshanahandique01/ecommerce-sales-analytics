# E-Commerce Sales & Customer Behavior Analytics

## Tech Stack
- PostgreSQL
- Python (Pandas, NumPy, Matplotlib, Seaborn, SQLAlchemy)
- Power BI

## Dataset
Brazilian E-Commerce Public Dataset (Olist) from Kaggle

## Project Overview
This project analyzes customer behavior, sales performance, payment trends, and delivery efficiency using SQL, Python, and Power BI.

## Key Features
- Data Cleaning and Transformation
- SQL Business Analysis
- Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard
- KPI Tracking and DAX Measures

## Dashboard Pages
1. Executive Summary
2. Sales Analysis
3. Customer Insights
4. Delivery Analysis
